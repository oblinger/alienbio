# alienbio - Alien Biology Architecture

Root architecture document for the Alien Biology system. The architecture is organized into three major subsystems.

- **[[infra]]** - Infrastructure: entity base classes, serialization, data management, configuration.
- **[[biology]]** - Biology: molecules, reactions, pathways, systems, organisms, and generators.
- **[[execution]]** - Execution: simulation engine, agent interface, experimentation framework.

- **[[Files]]** - Directory layout for the project.
