# BioMolecule

A molecule in the alien biology system.

## Properties

- **english_name** - human readable name
- **chemical_name** - formal chemical name
- **bdepth** - biosynthetic depth in the DAG
- **produced_by** - list of reactions that produce this molecule
- **consumed_by** - list of reactions that consume this molecule
