## Planned Documentation

The following modules exist in the codebase but documentation is pending:

- **Bio** — Core biology classes: molecules, reactions, compartments, simulation
- **CLI** — Command-line interface and command dispatch
- **Commands** — Individual CLI command implementations (run, report, expand)
- **Context** — Runtime pegboard connecting all subsystems
- **Entity** — Base class infrastructure for all biology objects
- **Generator** — Template expansion and synthetic biology generation
- **IO** — Serialization, parsing, and persistence
- **Protocols** — Protocol definitions and type contracts
